# React Audio Player

Audio Player using React, Vite, Tailwind as usual.


## Technologies Used

1. React
2. Vite
3. Tailwind

## How to run

Simply install the dependencies and run on your local host.
```bash
$ npm install
$ npm run dev
```

## Deployment

I deployed this app using Vercel if you want to test it.
